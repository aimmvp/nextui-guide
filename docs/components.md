# 업무 Components

## nu-btn

### >> SWING
![SWING BUTTON](/nextui-guide/images/components/swing_button.png)

### >> Next UI
```nu-btn``` component 는 __color__, __disabled__ 의 props 를 제공합니다.

#### props
 * __color__ : 1,2,3 의 옵션을 제공하며, defalut 값을 제공합니다.
 * __disabled__ : true , false 값을 제공합니다.

#### 예제
![NextUI Button](/nextui-guide/images/components/nu_button.png)

#### 예제소스
```js
<nu-btn>조회</nu-btn>
<nu-btn color="1">일반</nu-btn>
<nu-btn color="2">경고</nu-btn>
<nu-btn color="3">처리</nu-btn>
<nu-btn color="3" disabled="true">비활성화</nu-btn>
<nu-btn><v-icon>mdi-magnify</v-icon></nu-btn>
<nu-btn><v-icon>mdi-menu</v-icon></nu-btn>
```

::: tip
 Material Design Icon : [https://materialdesignicons.com/cdn/2.0.46/](https://materialdesignicons.com/cdn/2.0.46/)

 Vuetify Buttons : [https://vuetifyjs.com/ko/components/buttons](https://vuetifyjs.com/ko/components/buttons)
:::


## nu-input
### >> SWING
![SWING Input](/nextui-guide/images/components/swing_input.png)

### >> Next UI
```nu-input``` component 는 __hint__, __label__, __required__, __accent__, __disabled__, __readonly__ 의 props 를 제공합니다.


#### props
 * __hint__ : Hint Text 로 표시됩니다.
 * __label__ : input label 로 표시됩니다.
 * __reguired__ : true, false 의 옵션을 제공하며, default는 false 입니다.
 * __accent__ : true, false 의 옵션을 제공하며, default는 false 입니다.
 * __disabled__ : true, false 의 옵션을 제공하며, default는 false 입니다.
 * __readonly__ : true, false 의 옵션을 제공하며, default는 false 입니다.

#### 예제
![NextUI Input](/nextui-guide/images/components/nu_input.png)

#### 예제소스
```js
<nu-input hint="기본" label="기본"></nu-input>
<nu-input required="true" hint="필수" label="필수"></nu-input>
<nu-input accent="true" hint="강조" label="강조"></nu-input>
<nu-input disabled="true" hint="비활성화" label="비활성화"></nu-input>
<nu-input readonly="true" hint="readonly" label="readonly"></nu-input>
```

::: tip
 Vuetify Text fields : [https://vuetifyjs.com/ko/components/text-fields](https://vuetifyjs.com/ko/components/text-fields)
:::
