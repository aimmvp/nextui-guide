# 업무 Components

# nu-input

기본 텍스트 입력기능 이외에 추가적으로 display format 지정, 지정하지 않은 문자열 입력 불가, validator 정의 등의 기능을 설정 할 수 있다.

# >> SWING

# property

property Description type default
iClass 기본제공 Type 이며, 필수입력의 경우 pil, 강조입력의 경우 gang 을 입력한다. String 없음
placeholder 사용자에게 알려줘야 할 문구(hint) 등을 명시하며, 동작방식은 브라우저마다 상이할 수 있다. String 없음
disabled HTML의 disabled 속성과 동일한 기능을 제공한다. Boolean false
readonly HTML의 readonly 속성과 동일한 기능을 제공한다. Boolean false
itype 입력하는 데이터의 속성에 따라 표시 형식을 설정한다.[number, moneyWon, juminNum, foreignNum, password] String 'text'
text-align 텍스트 정렬기준을 설정한다.[right, center] String 'left'
label Component 의 label을 입력한다. String 없음

# 예제

<nu-input ></nu-input>
<nu-input iclass="pil" placeholder="필수입력입니다."></nu-input>
<nu-input iclass="gang" placeholder="강조입력입니다."></nu-input>
<nu-input disabled="true" label="비활성화입니다."></nu-input>

# nu-select (V)

Select Box 형태로 리스트에서 1개 ~ N개 의 항목을 선택 할 수 있다. 필수입력, 강조입력, 선택옵션, 전체선택옵션, 텍스트정렬, 리스트정렬, 리스트갯수 지정

# >> SWING

# property

property Description type default
iclass 기본제공 Type 이며, 필수입력의 경우 pil, 강조입력의 경우 gang 을 입력한다. String 없음
disabled HTML의 disabled 속성과 동일한 기능을 제공한다. Boolean false
chooseOption 최초 선택 값으로 label은 -선택-, value는 "" 으로 설정한다. Boolean true
multiSelect 선택항목의 다중 선택 가능여부를 선택한다. 선택 시 checkbox 를 통해 다중값 선택이 가능하다. Boolean false
allOption 전체 항목을 표시한다. label은 -전체-, value는 all 로 설정된다. 단, multiSelect 가 "true" 인 경우만 선택 가능하다. Boolean true

To-Do

property Description type default
chooseOptionLabel 최초 선택 값의 label을 설정한다. String 없음
sort_options 항목 리스트의 배열기준 값을 설정한다.[label, value] String 'label'
sort_method 항목 리스트의 배열순서(오름차순, 내림차순)를 설정한다.[asc, desc] String 'asc'
visible_row_num selectbox에 보여줄 리스트 항목 갯수를 설정한다. Number 5

# 예제

<nu-select ></nu-select>
<nu-select chooseOption="필수입력" iclass="pil"></nu-select>
<nu-select chooseOption="강조입력" iclass="gang"></nu-select>
<nu-select chooseOption="비활성화" disabled=true></nu-select>

# nu-textarea (V)

멀티라인 형태로 입력 뷰를 제공해서 긴 텍스트를 입력받는 용도로 사용한다. 개행이 필요한 경우 엔터키를 입력해서 개행처리하며, 입력한 텍스트가 영역을 초과한 경우 자동으로 스크롤이 활성화된다.

# >> SWING

# property

property Description type default
iclass 기본제공 Type 이며, 필수입력의 경우 pil, 강조입력의 경우 gang 을 입력한다. String 없음
label Textarea 의 label을 입력한다. String 없음
disabled HTML의 disabled 속성과 동일한 기능을 제공한다. Boolean false
maxLength 최대 입력값을 설정한다. number 100

# 예제

<nu-textarea label="기본 Textarea"></nu-textarea>
1

<nu-textarea iclass="pil" label="필수 Textarea"></nu-textarea>
1

<nu-textarea maxLength="19" label="최대입력값(19) Textarea"></nu-textarea>
1

<nu-textarea disabled="true" label="비활성화 Textarea"></nu-textarea>
1

# nu-radio-group / nu-radio(v)

Radio 는 1개의 항목을 체크하는 용도로 사용된다. nu-radio-group 과 nu-radio 로 이루어집니다.

# >> SWING

# property

nu-radio-group

property Description type default
alignMode Radio 배열 모드를 지정한다. [col, row] String col
disabled HTML의 disabled 속성과 동일한 기능을 제공한다. Boolean false

nu-radio

property Description type default
label radio 의 label 로 표시된다. String 공백
value radio 의 value 로 정의된다. String 공백

To-Do

property Description type default
init-idx Radio 버튼의 초기 설정 값 index 를 설정한다.(첫 번째 값 : 0) Number 0

# 예제

// 세로(col)배열
<nu-radio-group>
  <nu-radio label="세로1" value="radio-1"></nu-radio>
  <nu-radio label="세로2" value="radio-2"></nu-radio>
  <nu-radio label="세로3" value="radio-3"></nu-radio>
</nu-radio-group>

// 가로(row) 배열
<nu-radio-group alignMode="row">
  <nu-radio label="가로1" value="radio-1"></nu-radio>
  <nu-radio label="가로2" value="radio-2"></nu-radio>
  <nu-radio label="가로3" value="radio-3"></nu-radio>
</nu-radio-group>

// 라벨없음
<nu-radio-group>
  <nu-radio label="라벨없음" value="radio"></nu-radio>
</nu-radio-group>

# nu-checkbox-group / nu-checkbox(v)

Checkbox 는 1개 또는 N개의 의 항목을 체크하는 용도로 사용된다. nu-checkbox-group 과 nu-checkbox 로 이루어집니다.

# >> SWING

# property

nu-checkbox-group

property Description type default
rows 가로배열 component 수를 정의한다. rows=1 인 경우 1행에 모두 표시한다. Number 없음
cols 세로배열 component 수를 정의한다. cols=1 인 경우 1열에 모두 표시한다. Number 1
disabled HTML의 disabled 속성과 동일한 기능을 제공한다. Boolean false
init-idx Checkbox 버튼의 초기 설정 값 index 를 설정한다.(첫 번째 값 : 0) Number 0

nu-checkbox

property Description type default
label checkbox 의 label 로 표시된다. String 공백
value checkbox 의 value 로 정의된다. String 공백

# 예제

<nu-radio-group>
  <nu-radio label="세로1" value="radio-1"></nu-radio>

	<nu-radio label="세로3" value="radio-3"></nu-radio>
</nu-radio-group>

  <nu-radio label="가로2" value="radio-2"></nu-radio>

# nu-inputCalendar

달력에서 날짜나 월을 선택하여 입력하는 용도로 사용한다.

# SWING

# property

property Description type default
iClass 기본제공 Type 이며, 필수입력의 경우 pil, 강조입력의 경우 gang 을 입력한다. String 없음
valueType 캘린더의 형태와 날짜 표현방식을 선택한다. YMD의 경우 yyyy-MM-dd, YM의 경우 yyyy-MM 으로 표시한다. String 'YMD'
label InputCalendar 의 label 로 표시된다. String 공백

# 예제

<nu-inputCalendar iClass='gang' valueType="YMD" label="yyyy-MM-dd"></nu-inputCalendar>
<nu-inputCalendar iClass='pil' valueType="YM" label="yyyy-MM"></nu-inputCalendar>
